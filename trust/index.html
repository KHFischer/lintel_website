STANDARDS

Trust and standards alignment

Lintel Standard, Inc. is pre-seed. The company does not currently hold formal compliance certifications, and this page does not claim any. A future version of this page will list certifications, audit reports, and operational security documentation as they become available.

In the meantime, this page names the standards and specifications Lintel's architecture aligns with by design. These are named as references, not as endorsements or audit results.

REFERENCE SET

Aligned standards and specifications

  • IETF RATS Remote Attestation Procedures — the IETF working group's architecture for verifying claims from remote systems.
  • Confidential Computing Consortium Principles Industry reference framework for confidential-computing deployments.
  • AICPA SOC 2 Trust Services Criteria The criteria framework (Security, Availability, Confidentiality) informing Lintel's internal controls posture. Lintel is not SOC 2 audited at this time.
  • ISO/IEC 27001 control objectives Information-security control objectives referenced in Lintel's internal design. Lintel is not ISO 27001 certified at this time.
